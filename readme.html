Instalación: los famosos 5 minutos de instalación

  1. Descomprima el paquete .ZIP en un directorio vacío y cárguelo.
  2. Acceda al fichero wp-admin/install.php desde su navegador. Esto le llevará a través de un proceso para configurar un fichero wp-config.php con los datos de conexión a su base de datos.
    1. Si por alguna razón esto no funciona, no se preocupe. Esto no funciona en todos los alojamientos web. Acceda al fichero wp-config-sample.php con ayuda de un editor de texto como WordPad o similar y complete los datos de conexión a su base de datos.
    2. Guarde este fichero como wp-config.php y cárguelo.
    3. Acceda a wp-admin/install.php desde su navegador.
  3. Una vez que el fichero esté configurado, el instalador creará las tablas necesarias para su sitio. Si ocurre un error, revise su fichero wp-config.php, e intente nuevamente. Si vuelve a fallar, por favor, visite los foros de soporte con todos los datos que pueda aportar.
  4. Si no ha ingresado alguna contraseña, anote la contraseña que se le ha asignado. Si usted no ha proporcionado un nombre de usuario, éste será admin.
  5. Luego, el instalador le lleveará a la página de inicio de sesión. Ingrese con el nombre de usuario y contraseña que eligió durante la instalación. Si una contraseña fue generada para usted, entonces puede hacer clic en 'Perfil' para cambiarla.

Actualización

Utilizando el actualizador automático

Si está actualizando desde la versión 2.7 o superior, puede utilizar este actualizador:

  1. Acceda a wp-admin/update-core.php desde su navegador y siga las instrucciones.
  2. ¿Esperaba algo más? ¡Esto es todo!

Actualización manual

  1. Antes de actualizar, asegúrese de realizar copias de seguridad de cualquier archivo que haya modificado, como del fichero index.php.
  2. Elimine los ficheros de su anterior versión de WordPress, guardando previamente aquellos que haya modificado.
  3. Cargue los nuevos ficheros.
  4. Dirija su navegador a /wp-admin/upgrade.php.

Cambio en las plantillas de los temas

Si usted ha personalizado las plantillas de su tema, puede que necesite hacer algunos cambios en las versiones superiores.

Migrando desde otros sistemas

WordPress puede importar desde varios sistemas. En primer lugar usted necesita tener WordPress instalado y funcionando como se describió anteriormente, antes de usar nuestra herramienta de importación.

Requerimientos del sistema

  • PHP versión 5.2.4 o superior.
  • MySQL versión 5.0 o superior.

Recomendaciones del sistema

  • El módulo mod_rewrite de Apache.
  • Un enlace a http://wordpress.org en su sitio.

Recursos en línea

Si alguna de sus preguntas no tiene respuesta en este documento, le sugerimos que aproveche los numerosos recursos de WordPress en la red:

Interfaz XML-RPC y Atom

Puede publicar en su blog de WordPress usando herramientas como Windows Live Writer, Ecto, w.bloggar, Radio Userland (es decir, puede utilizar la función de correo-a-blog de Radio Userland), NewzCrawler, ¡y otras herramientas que soporten las API de publicación! :) Puede leer más acerca del soporte de XML-RPC en el Codex.

Publicar desde el correo

Puede publicar por medio de una cuenta de correo. Para configurar esta opción, vaya a la pantalla de opciones de "Escritura" y complete los datos de su cuenta POP3 privada. Luego necesitará configurar wp-mail.php para ejecutarlo periódicamente para revisar el buzón en busca de nuevos correos. Puede hacerlo mediante cron-jobs, o si su alojamiento no lo permite, puede buscar uno de los muchos servicios de monitorización web para que compruebe la URL de su wp-mail.php.

Publicar es sencillo: cualquier correo que envíe a la dirección especificada será publicada, con el asunto como el título. Lo mejor es mantener en privado la dirección. El script borrará los correos una vez publicados correctamente.
